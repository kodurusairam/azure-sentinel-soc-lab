# azure-sentinel-soc-lab
Microsoft Sentinel SOC Lab – SIEM deployment and threat hunting using KQL on Azure.

This project demonstrates SIEM deployment and threat hunting using Microsoft Sentinel on Microsoft Azure.

## Technologies Used
- Microsoft Azure
- Microsoft Sentinel (SIEM)
- Azure Data Explorer
- Kusto Query Language (KQL)

## Lab Setup
1. Created Azure Resource Group
2. Deployed Log Analytics Workspace
3. Enabled Microsoft Sentinel
4. Installed Sentinel Training Lab
5. Connected Data Sources

## Threat Hunting Queries

### Brute Force Login Detection
Detects abnormal authentication attempts.

### Suspicious Process Execution
Detects PowerShell and command execution patterns.

### Network Connection Analysis
Identifies unusual outbound connections.

### User Login Activity Analysis
Detects suspicious login behavior.
